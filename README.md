


##  Question 4 – Web Scraping CSV Data Extraction


## Description
This program scrapes hotel room data from two hotel booking websites( not real hotel booking websites but sample sites) and stores the
Extracted information in a CSV file for comparison.

## Data Extracted
- Hotel name
- Location
- Distance from downtown
- Room description
- Price
- Number of nights

## Libraries Used
- `requests` – to retrieve web pages
- `BeautifulSoup` – to parse HTML content
- `csv` – to store and read data from a CSV file

## Output
- A CSV file containing structured hotel room data  and to read from the CSV file and display in the terminal

 python 'Que. 4 .py'
